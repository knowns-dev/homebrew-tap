# Homebrew Tap for Knowns CLI

This is the official Homebrew tap for [Knowns CLI](https://github.com/knowns-dev/knowns).

## Installation

```bash
brew tap knowns-dev/tap
brew install knowns
```

Or in a single command:

```bash
brew install knowns-dev/tap/knowns
```

## Usage

```bash
knowns --help
kn --help  # shorthand alias
```

## Updating

```bash
brew upgrade knowns
```
